# Quests

This directory contains all quest and adventure information.

## Structure

- **active/** - Current, ongoing quests
- **completed/** - Finished quests
- **side-quests/** - Optional adventures

## Quest Management

### Creating a New Quest
1. Use `/templates/quest-template.md`
2. Save to `active/`
3. Include objectives, NPCs, locations, and rewards
4. Track progress as the party works on it

### Quest Status
- **Active**: Quest is available or in progress → `active/`
- **Completed**: Quest finished successfully → `completed/`
- **Side Quest**: Optional content → `side-quests/`

### Updating Quests
- Check objectives as they're completed
- Update quest status after each session
- Move to `completed/` when finished
- Link to session notes where progress was made

### Quest Organization Tips
- Use clear, descriptive quest titles
- Link to all related NPCs and locations
- Note quest dependencies (quests that lead to other quests)
- Track rewards and whether they've been claimed
- Include multiple solution paths when possible

## Quest Types

### Main Quests
Major storyline quests that advance the campaign's primary narrative.

### Side Quests
Optional adventures that provide character development, world-building, or additional rewards.

### Personal Quests
Quests tied to specific character backstories or goals.

### Faction Quests
Missions given by or related to specific factions.
