# Templates

This directory contains all templates used for creating consistent content throughout the campaign.

## Available Templates

### Core Templates

- **[character-template.md](character-template.md)** - For player characters and NPCs
- **[location-template.md](location-template.md)** - For cities, dungeons, and other places
- **[session-notes-template.md](session-notes-template.md)** - For recording session recaps
- **[session-planning-template.md](session-planning-template.md)** - For preparing upcoming sessions
- **[party-template.md](party-template.md)** - For party overview + state tracking
- **[party-member-abilities-template.md](party-member-abilities-template.md)** - For per-member ability pages (under a party folder)
- **[quest-template.md](quest-template.md)** - For adventures and missions
- **[item-template.md](item-template.md)** - For equipment and magic items
- **[creature-template.md](creature-template.md)** - For monsters and creatures (lore & reference)
- **[creature-combat-ready-template.md](creature-combat-ready-template.md)** - For NPCs and creatures you'll run in combat (battle-optimized)
- **[faction-template.md](faction-template.md)** - For organizations and groups

## Creature Template Choice

**Use `creature-template.md` for:**
- Generic/reference creatures
- Vanilla monsters from official books
- Creatures that won't see combat
- Lore entries and bestiary items

**Use `creature-combat-ready-template.md` for:**
- Custom NPCs you'll run in combat
- Boss fights and major encounters
- Creatures with tactics and behavior
- Anything designed for YOUR table

The combat-ready template prioritizes AC/HP/defenses at the top and relegates flavor to the bottom, so you can glance and play.

## How to Use Templates

1. **Copy the template** - Don't edit the template files directly
2. **Rename the file** - Use descriptive, kebab-case names
3. **Fill in the sections** - Complete all relevant sections
4. **Save to correct location** - Follow the directory structure
5. **Link related content** - Connect to other documents

## Template Sections

All templates include:
- **Tags** - For organization and searching
- **Metadata** - Dates, status, and key attributes
- **Quick Reference** - Important information at a glance
- **Detailed Sections** - Complete information
- **Related Links** - Connections to other documents

## Customization

Feel free to:
- Add sections as needed for your campaign
- Remove sections that aren't relevant
- Modify to fit your style
- Create new templates for unique content types

## Best Practices

- **Be Consistent** - Use the same template for the same type of content
- **Fill Completely** - Don't leave important sections blank
- **Update Regularly** - Keep information current
- **Cross-Reference** - Link related documents
- **Use Tags** - Make content searchable

## Creating Custom Templates

If you need a template for content not covered here:

1. Start with the most similar existing template
2. Modify sections to fit your needs
3. Save the new template in this directory
4. Document it in this README
5. Update `.github/copilot-instructions.md`
