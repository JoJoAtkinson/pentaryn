# Items

This directory contains all equipment, treasure, and magic items.

## Structure

- **magic-items/** - Enchanted equipment and wondrous items
- **artifacts/** - Legendary and unique artifacts
- **mundane/** - Regular equipment and gear

## Item Management

### Creating a New Item
1. Use `/templates/item-template.md`
2. Save to appropriate subdirectory:
   - Common to very rare magic items → `magic-items/`
   - Legendary items and artifacts → `artifacts/`
   - Non-magical items → `mundane/`
3. Include all properties, attunement requirements, and lore

### Item Organization
- Track current owner
- Note attunement status
- Link to where/how the item was obtained
- Include item history and lore

### Rarity Guidelines

| Rarity | Power Level | Typical Value |
|--------|-------------|---------------|
| Common | Minor utility | 50-100 gp |
| Uncommon | Useful bonus | 101-500 gp |
| Rare | Significant power | 501-5,000 gp |
| Very Rare | Major ability | 5,001-50,000 gp |
| Legendary | Campaign-defining | 50,001+ gp |
| Artifact | Unique, powerful | Priceless |

### Homebrew Items
- Clearly mark homebrew items
- Note any balance concerns
- Include DM notes about usage
