### Void Ray vs Disintegration Ray naming — FEEL-R114-01

The slice asks to "verify Void Ray (save-based) vs Disintegration Ray (attack-roll) naming is
unambiguous for the DM." It is **not** unambiguous. Three distinct confusion vectors:

**Vector 1 — Name similarity at the table:**  
Both end in "Ray." Under time pressure (6-second round), DM reaches for memory:  
- "I want to use the legendary ray" → which one? Disintegration Ray is the more vivid name.  
- Correct answer: Void Ray is the LA; Disintegration Ray is the main action.  
- Easy to swap, especially because Disintegration Ray *also* appears to be more powerful (45 avg vs 22 avg).

**Vector 2 — DB verb collision (BUG-R106-01):**  
DB entries for `disintegration_ray` (main action) include verb `"ray"`.  
DB entry for `void_ray_legendary` (LA) includes verb `"force ray"`.  
DM typing "ray" at the GUI → hits `disintegration_ray` first (earlier in jsonl). If DM intended
the LA Void Ray, they get the wrong action.

**Vector 3 — Different mechanics for same descriptor:**  
DR = attack roll (+7 to hit, 45 avg force, perma-death rider, Recharge 5–6).  
VR = DEX save (DC 16, 22 avg force, no special rider, Legendary Action 2 cost).  
Using DR as an LA is impossible but the name similarity may cause DMs to assume they're related
(e.g. "the weaker version of Disintegration Ray fires off the LA pool").

**Recommendation:** Rename the main action from "Disintegration Ray" to "Disintegration Beam"
in beholder-thrulm.md, and update the `disintegration_ray` DB verbs to remove `"ray"`.  
This is a two-token change that eliminates all three confusion vectors.  
**Not auto-fixable — source file rename + DB entry update; needs interactive MCP + Joe's sign-off.**

### LA strategy observations (extends R113 findings)

When Sabriel has L3 slots, Drain Divinity (3 LA) dominates over Tentacle+Void Ray (3 LA) because
slot-stripping compounds across rounds. When all divine slots are exhausted, this rotation
(Tentacle+Void Ray) becomes the *only* viable 3-LA sequence that doesn't waste LA.

This means the beholder's LA behavior has a natural phase transition:
- **Phase 1 (Sabriel has L3/L2 slots):** DD every round. Move and Tentacle LA never used.
- **Phase 2 (Sabriel spell-exhausted or dead):** Tentacle + Void Ray cycle.

The current design has no LA variation in Phase 1 — it's single-path deterministic. A 2-LA cost
for DD would introduce actual choice (DD for 2 + Tentacle for 1, vs Tentacle for 1 + Void Ray for 2).
This extends FEEL-R113-02 (10th consecutive confirmation of DD dominance).
