## Phase B — Simulation: `legendary-action-loop`

Simulate 3 rounds of beholder LA economy against Compass Edge. Core question from open issue #21:
is Drain Divinity (3 LA) strictly dominated by Void Ray + Tentacle (2+1 LA) in practice?

### Setup Assumptions

- Beholder hovers at 40–60 ft, Void-Feeding active (+1 atk/dmg rolls, not DCs)
- Sabriel: divine martial (Paladin-like), assumed CHA +4 for saves, 2×3rd-level slots, 3×2nd-level
- Beholder actions tracked after its own turn each round

---

### Round 1 LA Economy

**Main action:** Multiattack (2 Tentacle + Maw) targeting Marwen

- Tentacle #1 vs Marwen AC 15: +9, needs 6+. P(hit) = 75%. Avg dmg = 0.75 × 14 = **10.5 bludgeoning + grapple attempt** (DC 16 STR/DEX check)
- Tentacle #2 vs Marwen AC 15 (advantage if grappled): P(hit) ≥ 75%. Avg dmg = **10.5 bludgeoning + deepen grapple**
- Maw vs Marwen (grappled → disadvantage on escape check, but Maw itself still +9): avg dmg = **21 (4d8+3+1 Void-Feeding) = 22 piercing**

Expected R1 main action damage vs Marwen: ~0.75 × (10.5 + 10.5 + 22) = **32.6 expected** (Marwen likely at 0 HP or near it after this turn)

**LA budget: 3. Beholder uses LA at end of each PC's turn:**

Option A — Drain Divinity on Sabriel (3 LA, costs entire budget for the round):
- Range: 30 ft. **Beholder must descend from 40-60 ft hover to within 30 ft to target Sabriel.**
- DC 16 CHA vs Sabriel CHA +4: needs 12+. P(fail) = 9/20 = **45%**.
- On fail: Sabriel loses her highest slot (3rd-level); beholder gains 6 THP.
- Actual strategic value: removes one 3rd-level divine smite (~5d8 radiant on a hit = ~22.5 radiant).
- Cost: beholder descends into melee threat range for Sabriel and Bazgar next turn.

Option B — Void Ray (2 LA) + Tentacle Lash (1 LA):
- Void Ray vs Marwen (if alive) or next target: DC 16 DEX (Marwen DEX +3 → needs 13+, P(fail) = 8/20 = 40%). Avg = 0.60 × 23 + 0.40 × 11.5 = **18.4 force** (with Void-Feeding +1 to dmg)
- Tentacle vs Bazgar AC 18: +9, needs 9+. P(hit) = 60%. Avg = 0.60 × 14 = **8.4 bludgeoning + grapple**
- Total expected: **26.8 damage, no descent required, beholder stays at height**

**Option A vs B:**
- Option A: 45% chance to strip 3rd-level smite (avg denied damage ≈ 0.45 × 22.5 = 10.1) + 6 THP for 45% of fires. Costs beholder its hover advantage; Bazgar can reach it next turn (AC 18 fighter with GWM potential).
- Option B: 26.8 expected damage + grapple threat. Beholder stays safe.

In Round 1, Option B is strictly better unless the beholder has already closed (via Shrine Drift bonus action). The cheat sheet's tactic note says beholder "hovers above altar (40-60 ft up)" — DD requires closing to 30 ft, which is not documented as a trade-off.

### Round 2 LA Economy (Marwen at 0 HP or retreated)

Party state: Marwen down or fled, Bazgar at ~49 HP, Sabriel at ~44 HP.

If beholder used Option A in R1 and descended: it is now within 30 ft of Sabriel and Bazgar.
- Bazgar GWM attack vs AC 17: d20+5-5 = d20+0 vs 17, needs 17+. P(hit) = 20%. Still non-trivial.
- Sabriel attacks vs beholder AC 17: divine martial likely +6 to hit, needs 11+. P(hit) = 50%.

If beholder used Option B in R1 and stayed high: it is safe, uses Main Action (Disintegration Ray if recharged: 33% → 33% chance it fires on R2).

**R2 LA — now Drain Divinity value changes:**

Sabriel (if DD hit R1): down to 1st/2nd-level slots. Highest remaining = 2nd level.
- DD success → beholder gains 4 THP for 3 LA. **Clearly suboptimal vs 26.8 from Option B.**

Sabriel (if DD missed R1 or not used): still has 3rd-level slot.
- DD still has 45% to strip 3rd-level, but beholder is already committed to descending pattern.

**Finding:** After round 2, Drain Divinity's value per LA drops by ~40% (from "deny 22.5 smite" to "deny 11 smite and gain 4 THP"). Void Ray + Tentacle remains stable at ~26 damage per round. **There is no in-stat-block trigger that tells the beholder to stop using DD** — a DM or AI running the beholder by the tactic notes could waste 3 LA every round for diminishing returns.

### Round 3 LA Economy

Sabriel's 2nd-level slots also exhausted. Remaining: 3-4 × 1st-level slots (spell points ≈ divine smite 2d8 = 9 avg).

DD on 1st-level slot: beholder gains 2 THP. **3 LA → 2 THP. Objectively useless.**

Meanwhile: Void Ray (2 LA) + Tentacle (1 LA) still averages ~26 damage. The beholder should have hard-switched to this LA mix by round 3 at the latest.

**Summary — LA dominant strategy:**

| Round | Ideal LA use | DD recommendation |
|-------|-------------|-------------------|
| R1 | Void Ray + Tentacle (stay high, deal damage) | Skip unless beholder already <30 ft |
| R2 | Void Ray + Tentacle (Marwen down; focus Sabriel/Bazgar) | Only if Sabriel still has 3rd-level slot AND beholder already descended |
| R3+ | Void Ray + Tentacle loop | Never (slot value too low vs damage opportunity) |

**LA economy is functional but has a documentation gap:** the tactic block says "use DD vs clerics/paladins" without specifying the range descent cost or the round-by-round value curve. A DM following the tactic note verbatim will over-invest in DD and expose the beholder unnecessarily.

---

### Key Findings

**FEEL-R310-01: DD range (30 ft) forces beholder descent — undocumented trade-off**

The beholder's default hover position is 40-60 ft above the altar. Drain Divinity has a 30-ft range. To use DD on any party member, the beholder must descend to within 30 ft of the target. This exposes it to:
- Melee attacks from Bazgar (GWM-capable; if crits occur, significant damage)
- Divine smite from Sabriel at close range

The cheat sheet and tactic block say nothing about this trade-off. DMs running the beholder as written will descend every turn DD is available, removing the encounter's intended tactical texture of "an aerial horror you can't easily reach."

**Recommendation for Joe:** Add a note to tactics (and cheat sheet) explicitly stating: "Use DD only if beholder has already descended for other reasons (e.g. Void Scream forced it in, or Shrine Drift passed through to reposition). Do not descend solely for DD; it negates the fly advantage."

**FEEL-R310-02: DD value collapses after round 2 — no internal trigger to switch**

Drain Divinity's value per LA drops from ~10 denied damage (R1) to ~3.5 denied damage (R3) as Sabriel's slots deplete. The stat block has no trigger, DC, or note indicating when the beholder should stop using DD. A DM running the encounter in rounds 3-6 (the likely actual combat length) will often use 3 LA for 2 THP + deny a 1st-level smite — a terrible exchange when Void Ray + Tentacle could deal 26 damage instead.

**Recommendation for Joe:** Add to DD's description (or the cheat sheet): "Only use if target has 2nd-level or higher spell slots remaining. If target's highest available slot is 1st level, switch to Void Ray + Tentacle instead."
