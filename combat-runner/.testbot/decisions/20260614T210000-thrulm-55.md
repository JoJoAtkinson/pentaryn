### New findings

**FEEL-F55-01 (NEW): UG → Multiattack same-turn combo eliminates PCs before resource expenditure**

The UG (lair init 20) → beholder Multiattack (TL×2 advantage + auto-crit Maw) sequence fires on the same "round" before the prone PC's turn. This creates a one-turn, ~66-avg-damage nova that exceeds both Bazgar (49 HP) and Sabriel (44 HP) single-turn HP.

The design intent for prone+advantage is typically: lair sets up the prone → PCs get a turn (stand up, counterattack) → beholder capitalizes next round. The current encounter squashes this two-round arc because:
1. Lair fires at init 20 (before most PCs)
2. Beholder has high initiative (+3 DEX = avg 13.5) and typically acts before a STR-primary fighter (Bazgar DEX −1 = avg 9.5)
3. Auto-crit Maw fires on the SAME turn as both TL hits — there's no window for the grappled target to escape

**Feel comparison with design intent:**
| Expected (two-round arc) | Actual (one-round nova) |
|---|---|
| R1: UG → Bazgar prone → Bazgar stands up (spends movement), swings at beholder | R1: UG → prone → TL×2 → Maw crit → Bazgar downed before turn |
| R2: Beholder TL (adv, Bazgar doesn't stand) → Maw crit | — (skipped entirely) |
| Bazgar uses Action Surge R1 | Never fires |
| Drama: Bazgar fights through the grab | Drama: Bazgar is immediately a body |

**Recommendation (log only — design call):** Beholder tactics note: "When UG has been used this round (lair action), prefer Void Scream or Disintegration Ray as main action — do NOT multiattack the prone PC in the same round. The intended arc is: Lair (set up prone) → PCs act (stand or don't) → beholder multiattack next round (advantage if still prone). Firing UG + Multiattack same-turn on the same target collapses player agency; the PC never contributes."

*Alternatively: this is a design choice — a CR 13 boss is supposed to obliterate level-5 parties. The "one round before you act" feel may be intentional. Joe decides.*
