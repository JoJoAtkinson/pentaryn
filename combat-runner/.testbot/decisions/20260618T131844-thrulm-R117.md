### New bug found: Shrine-Touched Perception skill inconsistency

**BUG-R117-PERC-01 (new):** `shrine-touched-derro.md` — Perception +1 is not derivable from stated stats.

- WIS 8 (modifier −1), Proficiency Bonus +3 (explicitly stated in stat block)
- Proficient Perception: −1 + 3 = **+2** (not +1)
- Non-proficient: −1 (not +1)
- Half-proficiency: −1 + 1 = 0 (not +1)
- Passive Perception 11 = 10 + 1 is internally consistent with +1, but the +1 doesn't derive from the stated modifiers

Root cause: appears to be a legacy calculation using PB +2 (−1 + 2 = +1), from before the explicit PB field was set to +3. Compare Stealth +5 = DEX +2 + PB +3, which correctly uses PB +3. Only Perception was not updated.

**Not auto-fixable.** Requires design decision: if proficient in Perception, correct value is +2 (passive 12); if non-proficient, −1 (passive 9). Log only.

### Phase B — New feel notes

**FEEL-R117-01 (new):** Lair action tier list is lopsided and deterministic. Void Eruption EV (~16–19 avg) is 4–5× Manifest Thralls (~3.6 avg) and provides quantifiable pressure Manifest Thralls cannot. Once party melee closes (R2+), the correct lair action pick requires zero deliberation. This mirrors the Drain Divinity dominant-strategy problem (FEEL-R116-02): the beholder's toolkit has dominant strategies that remove meaningful DM decisions. If Manifest Thralls is ever intended to be chosen, it needs a complete redesign (existing open item carried forward).

**FEEL-R117-02 (new):** Lair action R1 is frequently wasted. Initiative count 20 fires before most PCs have moved (unless the encounter starts in-progress or Bazgar has already crossed to melee range). If party positions are at engagement range when initiative is rolled, R1 Void Eruption hits; if they're at the entrance, it misses everyone. The stat block doesn't address this edge case. Recommend: add a note that Void Eruption triggers optimally R2+, or allow the beholder to hold its lair action (non-standard but DM-friendly).

**FEEL-R117-03 (new):** Void Eruption's sustained output (~16–19 avg every initiative 20) is meaningful pressure on the melee PCs over 3–4 rounds. Sabriel (HP 44) takes ~33 cumulative lair action damage by R3 on average fails, putting her at critical HP without any main action needed. The lair actions effectively serve as a "melee tax" that discourages Bazgar and Sabriel from staying near the shrine — but the shrine is the beholder's home ground and the dramatic center of the encounter. This creates a mild counterintuitive push: the party should fight the beholder at the shrine, but staying there is punishing. This tension is probably intentional design but worth noting in the tactics section.
