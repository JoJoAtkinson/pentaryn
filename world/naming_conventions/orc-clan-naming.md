# Orc Clan Territory Names — Language Conversion Rules

**Goal:** Transform Common (English) territorial descriptions into harsh, guttural orc language that *suggests* the meaning without being a literal translation.

## Core Principle

Take the **concept** (red mountain, ash valley, iron marsh) and rebuild it using:

1. Harsh consonant clusters
2. Short, clipped vowels (a, o, u preferred over e, i)
3. Orc-specific sound patterns
4. Clan suffix that sounds natural with the territory word

---

## Step-by-Step Conversion Process

### Step 1: Identify the Core Concept

Break down the English territory into 2 parts:

- **Feature** (what it is): peak, valley, marsh, river, forest, plain
- **Descriptor** (what makes it distinct): red, ash, iron, stone, blood, black

Examples:

- "Red Peak" → red + peak
- "Ash Valley" → ash + valley
- "Iron Marsh" → iron + marsh

### Step 2: Convert Each Part to Orc Phonemes

#### Descriptor Conversions (what makes it distinct)

| English    | Orc Root   | Sound Logic                                           |
| ---------- | ---------- | ----------------------------------------------------- |
| Red        | Rak        | harsh 'r' + 'k' sounds (proto-root)                   |
| Ash        | Gash       | sibilant + harsh consonant                            |
| Iron       | Gron       | heavy metal sound                                     |
| Stone      | Drok       | hard 'k' and 'r'                                      |
| Blood      | Ghor       | guttural, primal                                      |
| Black      | Dak        | dark, clipped                                         |
| Fire       | Zar, Thrak | crackling sounds (alt: Thrak for major fire features) |
| Storm      | Thrum      | rumbling                                              |
| Shadow     | Skor, Zhar | sibilant + hard (alt: Zhar for deeper shadow)         |
| Bone       | Krul       | hollow, hard                                          |
| Frost      | Zhul       | cold, sharp                                           |
| White/Pale | Drul       | pale, ghostly                                         |
| Green      | Vrok       | living, growing                                       |
| Deep/Dark  | Mor        | deep, abyssal                                         |
| Wild       | Urg        | untamed, chaotic                                      |

#### Feature Conversions (geographic type)

| English       | Orc Root   | Sound Logic                            |
| ------------- | ---------- | -------------------------------------- |
| Peak/Mountain | -thok      | hard, tall sound                       |
| Valley        | -val, -gul | keep 'val' soft, or -gul for harshness |
| Marsh/Swamp   | -mar, -mog | wet, heavy (alt: -mog for deep swamps) |
| River         | -zug       | flowing sound                          |
| Forest        | -gar       | dense sound (proto-root)               |
| Plain/Field   | -dun       | open, flat                             |
| Hill/Ridge    | -kor       | rising sound                           |
| Canyon/Gorge  | -vorg      | deep, echoing                          |
| Lake          | -rog       | still water                            |
| Wasteland     | -nak       | barren, enduring                       |
| Crag/Cliff    | -drak      | sharp, jagged                          |

### Step 3: Combine and Add Clan Suffix

Formula: `[Orc Descriptor] + [Orc Feature] + [Optional Clan Suffix]`

#### Without extra suffix (territory IS the name)

- Red Peak → Rak + thok → **Rakthok**
- Ash Valley → Gash + val → **Gashval** or Gash + gul → **Gashgul**
- Iron Marsh → Gron + mar → **Gronmar**

#### With clan suffix (adds "-people of")

- Red Peak → Rak + thok + -gar → **Rakthokgar** (people of the red peak)
- Ash Valley → Gash + gul + -vor → **Gashgulvor** (holders of the ash valley)
- Iron Marsh → Gron + mar + -nak → **Gronmarnak** (those who endure the iron marsh)

### Step 4: Sound Check & Simplification

**Rules:**

1. **Avoid triple consonants** — if combining creates `Gronmnar`, simplify to `Gronmar`
2. **Prefer 2 syllables for base** — `Rakthok` > `Rakathokul`
3. **End on hard consonant** — `-k`, `-g`, `-r`, `-th`, `-z` sound strong
4. **Test if you can shout it** — clan names should be battle-cry worthy

## Advanced: Elision and Natural Flow

For clans that have been named for generations, sounds naturally blend:

- **Rakthok** might become **Rathok** (k + th elides)
- **Gashgul** stays clean (sh + g works)
- **Gronmarnak** might become **Gromnak** (nm elides, ar→o for flow)
- **Ghorzug** stays (already smooth)

**Elision rule:** If two hard consonants clash awkwardly when shouted, drop one or insert a connecting vowel (usually `o` or `a`).

## Quick Generation Template

1. **What's the territory feature?** (peak, valley, marsh, river, etc.)
2. **What makes it distinct?** (color, element, quality)
3. **Look up both in the conversion tables above**
4. **Smash them together:** Descriptor-root + Feature-root
5. **Optional: Add clan suffix** (`-gar`, `-vor`, `-nak`, `-gath`)
6. **Sound check:** Can you shout it? Does it sound orcish?

## Clan Suffixes (Optional Add-Ons)

Use these when you want to emphasize "the people of" rather than just the place:

| Suffix | Meaning          | Usage                                                  |
| ------ | ---------------- | ------------------------------------------------------ |
| -gar   | people/clan      | **Rakthokgar** (people of the red peak)          |
| -vor   | holders/keepers  | **Gashgulvor** (holders of the ash valley)       |
| -nak   | those who endure | **Gronmarnak** (those who endure the iron marsh) |
| -gath  | gathering/horde  | **Krulkorgath** (gathering at bone hill)         |
| -gul   | band/pack        | **Gashgul** (pack of the ash valley)             |

## Cultural Notes

- **Western Orcs (Garrok Confederation):** May use Common alongside orc names in treaties: "Rakthok Clan" or "the Rakthok"
- **Eastern Orcs (Rakthok Horde):** Rarely translate; you learn the orc name or you don't learn it at all
- **Mixed Clans:** Sometimes hyphenate: "Ironmarsh-Gromnak" (Common-Orc hybrid for diplomacy)

---

## Proto-Root Connection

Remember: Orc names mutate the ancient proto-root `ar` → `gar` or `rak`.

- If you want to show ancient lineage or sacred territory, include `gar` or `rak` in the name
- Example: **Rakthok** (rak = proto-root + thok = peak) suggests this peak has ancient significance
- Example: **Dakgar** (dak = black + gar = people/proto-root) implies ancient forest guardians
