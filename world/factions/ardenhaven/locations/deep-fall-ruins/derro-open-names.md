# Derro Open Names (Dwarven-Style)

Derro in the Deep Fall Ruins use **dwarven-style** naming (hard consonants, short vowels, weighty endings) — but the sound should feel **dark**, **mean**, and **unwelcoming**.

Use these as “grab-and-go” names for hostile derro NPCs.

Named NPC ready for combat: [Dulgor-grum](./npcs/dulgor-grum.md).

## Names

- **Khardruk-dun** (KHAR-druck-dun) — *rusted*
- **Grumkhor-hammar** (GROOM-kor-HAM-mar) — *hammering*
- **Drakmor-dur** (DRAK-mor-DOOR) — *grave*
- **Thar'brum** (THAR-broom) — *judging*
- **Khorbad-dum** (KOR-bad-dum) — *sour*
- **Bardrog-dur** (BAR-drog-DOOR) — *iron-fanged*
- **Dulgor-grum** (DULL-gor-grum) — *pit-deep*
- **Brakthar-kar** (BRAK-thar-kar) — *chainbound*
- **Drogbar-bor** (DROG-bar-bor) — *blood-oath*
- **Thargrum-dun** (THAR-grum-dun) — *smoldering*
