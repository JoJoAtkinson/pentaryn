### Feel issues / DESIGN DECISIONS

**FI-BEL3-37-A — DD-3LA-monopoly-37th-cycle-confirm:**
All 3 legendary actions went to Drain Divinity every round Sabriel had L2+ slots (all 3 rounds). Result: beholder applied zero additional pressure via LA (no Void Ray on 8-HP Marwen, no Tentacle on 13-HP Bazgar). The trade was slot attrition vs. immediate kills — and the beholder chose attrition each time, correctly per tactics. But the feel consequence is: the LA phase is one repeated action (DD) until Sabriel's high-level slots run out. Same issue flagged in FI-EV7-28-A (empty-void, 28th cycle) and MQ-BEL3-33-A. **Recommend:** DD at 2 LA cost (freeing 1 LA for Tentacle/Void Ray per round), or explicit tactics note: "if target has only L1 slots remaining, switch LA to Void Ray + Tentacle instead of DD."

**FI-BEL3-37-B — First beholder-retreat outcome for slice #3 (37th cycle):**
All prior BEL3 cycles (31st, 32nd, 33rd) ended in TPK. This cycle ended in beholder retreat. Key differences: (1) Sabriel was never threatened directly by beholder's main action, (2) all three Divine Smites hit (22+14+16 = 52 radiant damage unresisted), (3) VS never recharged (zero AoE disruption). The "limited" designation (no Disintegration Ray) appears appropriately less lethal — the party survived but is heavily resource-depleted.

**FI-BEL3-37-C — Sabriel untouched (44/44 HP) while contributing 52 radiant damage:**
