## Findings

### Bugs auto-fixed

1. **MQ-BEL25-A** (25th-cycle confirm, now fixed): `compel_thrall` used on a dead/downed target. Beholder issued compel_thrall (bonus) targeting TB to attack Marwen — but Marwen was dropped to 0 HP on the same beholder turn by TL2-redirect, making the compel meaningless when TB's turn arrived. The .md Compel Thrall targeting section lacked a live-target guard. **Fix applied:** added note to beholder-thrulm.md line 51 (Compel Thrall targeting bullet): *"Skip if no live thralls (MQ-BEL25-A): Do not use compel_thrall if all thralls are dead or at 0 HP — there is no valid charmed target and the bonus action is wasted. Use shrine_drift instead."*

### Feel issues / DESIGN DECISIONS

- **FI-BEL26-A** — VS R1 dominance (25th-cycle confirm): Both Bazgar and Sabriel failed VS saves (rolls 10 and 7 vs DC 16). 40 psychic each drove Bazgar to 9/49 and Sabriel to 4/44. This made R2 trivially lethal. Beholder took 0 damage this run. Pattern: when VS lands on 2+ PCs at R1, the fight is over in 2 rounds. Fear rider (FI-149) worked as intended — both frightened PCs attacked thralls, not beholder. No counterplay for level-5 party vs 6d10 psychic. **Intent confirmed**: this is expected for a CR 13 beholder vs level-5 party. No change recommended.

- **FI-BEL26-B NEW — "compel wasted same-turn kill"**: Beholder's TL2 redirect killed Marwen (TL1 killed Sabriel) in the same action sequence as compel_thrall was issued. The compel was mechanically correct *at issue time* (Marwen was alive when the bonus action resolved conceptually), but the thrall's turn came after Marwen was already down. This is a timing interaction gap: beholder multiattack can kill the compel target before the thrall acts. .md fix above (MQ-BEL25-A) handles the "no live thralls at all" case; the "target just died to the beholder's own attack" case is a subtler pattern — log for human review. Recommendation: beholder should use `shrine_drift` as bonus when multiattack is targeting remaining PCs (likely lethal), reserving compel for rounds when a thrall can act *before* the beholder's multiattack cleans up.

- **MQ-BEL25-B CONFIRM (26th iteration, still open)**: VS start state for slice #3 is undocumented in the slice description (briefing or _overview.md). All 25 beholder-escorts-limited cycles have treated VS as available at start of combat. This assumption should be formally documented. Log to DESIGN DECISIONS.

- **FI-BEL26-C — No-DR fight is still 2-round TPK**: Slice #3 restricts DR to probe the legendary-economy without the disintegration ace card. Even without DR, VS + MA + VR(LA) = TPK R2. This is the 3rd consecutive no-DR TPK-in-R2. The party has no answer to VS opening play. Beholder took 0 damage.
