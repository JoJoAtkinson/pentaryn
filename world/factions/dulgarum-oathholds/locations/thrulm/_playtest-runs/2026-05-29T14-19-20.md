### Mechanical questions

**MQ-EV8-A: Lair action DCs differ between DB and simulation (sim harness error — no spec change)**
- `void_eruption` DB output specifies "DC 16 Dex save"; simulation used "DC 14 Con". `unstable_ground` DB output specifies "DC 16 Dex save"; simulation used "DC 15 Str". Both are sim harness errors introduced this cycle. The DB specs are authoritative. At DC 16 vs L5 PCs (+1/+0/+0 Dex), lair actions would land on ~75% of rolls (vs ~65% at DC 14). This makes R2/R3 lair effects slightly more dangerous than simulated. Net effect: TPK would arrive ~0.5 rounds earlier with correct DCs. No spec change needed.

**MQ-EV8-B: Void Scream simulation applied "save = 0 damage" but DB specifies "savers take half" (sim harness error — no spec change)**
- The DB output explicitly states "savers take 18" (half of 36). Simulation coded `pass = 0 damage`. Sabriel passed R1 VS (roll 20, DC 16) and took 0 instead of 18 necrotic. Corrected in run notes above (Sabriel 26/44 end R1, not 44/44). With correct HP, the fight likely ends R4 instead of R5 — same outcome, slightly faster. No spec change. Harness should parse dispatch output for save damage rather than re-implementing save resolution logic.

**MQ-EV8-C: Sim modeled Sabriel casting Cure Wounds (self-damage hazard trigger) — should model non-healing actions (sim methodology note)**
- The simulation had Sabriel fall back to Cure Wounds after smites were exhausted, triggering the chamber hazard and dealing self-damage. A competent player would NOT do this after the first backfire. Correct model: Sabriel should use Guiding Bolt (L1) or Thunderwave when forced to offensive spell use, and fall to basic attacks + LoH when L1 slots are depleted. This is a sim methodology note, not a spec bug. The Cure Wounds self-damage was retained in the sim narration to document the hazard, but the HP tracking above uses the corrected Guiding Bolt path.

**MQ-EV8-D: Void Ray damage on unconscious targets — auto-fail Dex save confirmed**
- Unconscious creatures automatically fail Strength and Dexterity saving throws (PHB). VR is a DC 16 Dex save. Therefore unconscious Bazgar and Sabriel take full VR damage (not half). Applied in run above. This is consistent with existing rulings. No spec change.
