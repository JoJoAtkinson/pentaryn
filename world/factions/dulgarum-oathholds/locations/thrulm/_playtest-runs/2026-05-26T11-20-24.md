### Feel issues / DESIGN DECISIONS

- **FI-125 (NEW — feel): Sabriel LOH revival loop dominates R3-R8.** Sabriel spent her action every round from R3 to R8 reviving Marwen with Lay on Hands (1 HP each time). She never attacked the beholder or used any aggressive ability after R1. The fight resolved as: Beholder kills Marwen (via legendary VR after Bazgar's turn), Sabriel revives, repeat. The beholder eventually wins by killing Sabriel directly (R8) and then Marwen stays dead. **Feel assessment:** The loop is mechanically correct (LOH is void-safe, costs 1 HP pool point, costs her action — all rules working) but produces 6 identical turns from the divine martial. At table the DM should narrate this loop as escalating desperation, but the absence of healing alternatives (Sabriel's L2 slots are gone from Drain Divinity + R1 smite) forces this pattern. **Recommend adding tactic note to beholder.md: "If Sabriel appears to be cycling LOH on a downed ally, burn 1 legendary (Tentacle) on Sabriel per her turn to threaten her — LOH is a class feature and the beholder cannot stop it, but threatening Sabriel may force her to use LOH defensively on herself instead."** Do NOT auto-fix; this is a tactics annotation.

- **FI-126 (NEW — sim harness): Frightened condition not re-saved each turn.** Sim applies frightened as permanent once failed. At-table Bazgar gets an end-of-turn Wis save (DC 16, +0 Wis, ~20% pass rate). Expected frightened duration: 3-5 more rounds before clearing. Actual sim: frightened R2 through R8 = 6 rounds, slightly overstated but directionally correct. Fix in sim script next cycle.

- **FI-127 (NEW — design choice): Void Scream fired once in 8 rounds (R2 only).** Recharge-6 = 16.7% chance per turn. After R2 fire, VS never recharged in 7 more turns (rolls: 5,2,5,2,2,1,5,4 — never a 6). The fight was defined by R2's VS (Bazgar frightened, 22 psychic; Sabriel takes 17) and then 6 rounds of attritional Void Ray spam. **Design question: is VS intended to be a once-per-fight swing (recharge-6, ~50% chance in 8 rounds = fires ~0-1×) or a recurring threat (recharge 5-6 = 33% per turn, fires ~1-2× per fight)?** Current feel: VS creates a single dramatic moment then disappears. If the design intent is sustained fear pressure, recharge 5-6 achieves that without making VS cheap. DESIGN DECISION — log for human.

- **FI-128 (NEW — feel): Drain Divinity becomes dead weight after R1.** DD fired once (R1, strips Sabriel's last L2). After that, only L1 slots remain on all PCs — DD on L1 gives +2 temp HP and costs 3 legendary actions. Void Ray (2 LA) does avg 22 damage. DD on L1 is objectively wrong; the sim correctly drops to VR. **Tactics note gap:** the current beholder.md Tactics section says "burn legendary on Drain Divinity rather than Void Ray" for clerics/paladins, but doesn't specify "stop when only L1s remain." Recommend adding: "Drain Divinity is worth spending on L2+ slots (returns +4-6 temp HP; cripples healing economy). Once the target's highest remaining slot is L1, prefer Void Ray instead." Low-risk documentation fix; can auto-apply next cycle if human approves.

- **FI-129 (NEW — feel): Lair action monotony post-thrall-wipe.** Once all thralls died R1, the lair action was Void Eruption 4× in a row (R2-R5), then Unstable Ground 3× (R6-R8). No variety. At-table the DM should be reminded to rotate: Unstable Ground when a PC is grappled (Bazgar R3-R6), Manifest Thralls if beholder wants to delay (below 60 HP). **Recommend adding a lair-action priority note to beholder.md start-of-turn checklist:** "Lair action rotation guidance: (1) Void Eruption if 2+ PCs within 20 ft of altar; (2) Unstable Ground on most mobile threat (Bazgar) if he's the last standing melee PC; (3) Manifest Thralls only if beholder HP > 70 and wants to buy time. Avoid running the same lair action 3+ rounds in a row." Documentation fix; can auto-apply if human approves.

- **FI-56 (3RD CONFIRM — sim harness, existing finding): Maw autocrit on grappled target not enforced in sim.** R6: Tentacle1 grapples Bazgar, then Maw lands (24 vs AC18) — should be autocrit (double 4d8+3 ≈ 47 avg). Sim applied 19 piercing instead. Bazgar survived 3 extra rounds because of this. At-table this would have killed Bazgar R3 (15 HP → autocrit Maw avg 47 → instant death + potentially grievous wound). **This is a harness issue, not a DB bug — DB spec correctly states "If the target is grappled by the beholder, this attack is a critical hit instead." The real encounter is MORE deadly than the sim shows. FI-88 (2nd cycle) escalated this to DD-11 URGENT — this is the 3rd confirm. Human sign-off still pending on DD-11.**
