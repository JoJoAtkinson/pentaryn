## Findings

### Sim errors (infrastructure, not spec bugs)

- **SIM-FC5-A (stagger rule check timing)**: STD-A incorrectly fired Ancient Resonance in R1. The stagger rule says evaluate at START of round; my sim checked at time-of-action. By STD-A's turn (init 7), STD-B had already set `std_ar["STD-B"] = True`, so the "both have AR" check returned false. Correct play: STD-A multiattacks in R1. Net effect: ~10 HP more on Bazgar in the corrected run vs sim; party would still TPK but Bazgar survives slightly longer in R1. **Not a spec change — DM guidance in .md is correct ("evaluated at START of round").**
- **SIM-FC5-B (dual dice)**: NPC tool rolls dice internally AND sim re-rolls with local random for HP tracking. Displayed narration and HP tracking use different numbers (e.g., DR shows 29 force via NPC tool but 48 force via local dice). Outcome unchanged for kills but numbers are inconsistent. Persistent issue from previous runs — needs infra fix in future.
- **SIM-FC5-C (lair action on downed target)**: Sim targeted Bazgar (downed, 0 HP) with Unstable Ground R2. The .md specifies skip unstable_ground if target is unconscious. Lair action was wasted. Correct play: rotate to `void_eruption` or `manifest_thralls`.

### Bugs auto-fixed

- None. All 29 action specs executed cleanly in Phase A. No DB spec issues found.

### Feel issues / DESIGN DECISIONS

- **FI-FC5-A (R1 front-loaded burst)**: STD-B fires Ancient Resonance BEFORE the lair action (STD-B init 25 > lair at 20), dealing 22 total to Bazgar + Marwen before the beholder even acts. With the lair Void Eruption following immediately, Marwen (32 HP) is downed in the first rotation at R1. The beholder then fires DR → kills Marwen permanently at R2. The party has almost no counterplay window. This is INTENTIONAL for the final confrontation slice (expected TPK), but the feel is "death by attrition before you act" rather than "overwhelmed by superior force." **Log for human — not a fundamental balance change; the question is whether the R1 init run of STD-B → lair → thralls → STD-A is intended to feel this oppressive.** (If STD-B rolled below 20 on init, the lair fires first and STD-B comes after — much less punishing. The high-init STD is the difference.)
- **FI-FC5-B (altar zone fire suppression feel)**: Marwen's Fireball is her peak play. Within the altar zone, fire vulnerability is suppressed on STDs (they take half instead of double). The narration calls this out, but a live DM should announce this BEFORE the caster commits the slot, not after. The .md documents it clearly; the playtest confirms the suppression is working mechanically. **No spec change needed — add to DM pre-fight checklist note** (already in README/overview).
- **FI-FC5-C (Antireality feel — POSITIVE)**: Antireality blocked Sabriel's L2 Divine Smite attack (hit 19 ≤ AC+2 = 19, exactly on margin). This is a memorable, surprising moment — the beholder "dodges" a smite it couldn't fully avoid. The threshold check (estimate from avg, not result) makes it tactically interesting. Positive confirmation; no change needed.
- **FI-FC5-D (DR disintegration feel — POSITIVE)**: Marwen's permanent death via Disintegration fired correctly and cleanly. The log called it out immediately. At a live table this is a high-stakes moment; the .md is explicit about the ruling (FI-NEW-FC4-LOH). No spec change needed.
- **FI-FC5-E (LoH exhaustion pacing)**: Sabriel burned all 25 LoH HP across 3 rounds (R1: 10, R2: 10, R3: 5). By R3 she had no pool left. LoH pressure from the beholder's legendary Tentacle Lash forcing her to spend on herself (rather than allies) didn't fire because the TL missed. The "correct" TL pressure play (per .md FI-125) would have shortened Sabriel's effective LoH pool further. The pacing felt tight and earned.

### Mechanical questions

- **MQ-FC5-A (stagger rule DM reminder)**: The .md correctly says "stagger evaluated at start of round before any actions resolve." This is easy to forget at the table — by the time the lower-init STD acts, the higher-init STD's `ar_used` flag is already set, which looks like "only one had AR." Recommend adding a ⚑ reminder bullet to shrine-touched-derro.md start-of-turn checklist: "At round start, BEFORE anyone acts: if BOTH STDs have AR available, mark the lower-initiative STD as 'AR HELD this round'." **Human decision: is a checklist addition appropriate, or is the ⚠️ note in Tactics section sufficient?**
