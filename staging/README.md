# Staging

Scratch area for drafts and work-in-progress that will be moved to a permanent home (e.g. `quests/active/`, `world/factions/...`, `characters/npcs/...`) once polished.

Anything in here is considered **draft / unsorted**. Do not link to files in `staging/` from canonical lore — links should point at the file's eventual home, not its temporary one.

When a file graduates, `git mv` it into place and update any references.
